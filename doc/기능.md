# 에디터 기능 (상단 메뉴)

Unity 에디터 상단 메뉴에서 실행하는 커스텀 기능 목록.
새 메뉴 기능을 추가할 때마다 이 문서에 항목을 추가한다.

---

## Tools ▸ World Exploration

### Import World Map (PNG → WorldMapData)

지형 분류 PNG + 고도 PNG를 읽어 게임이 사용하는 맵 데이터 에셋(`WorldMapData`)으로 변환(굽기)한다.

- **입력**:
  - `Assets/Game/Map/Authoring/world_terrain_1024x512.png` (지형 분류색: 심해/얕은바다/육지/산악)
  - `Assets/Game/Map/Authoring/world_height_1024x512.png` (고도 그레이스케일, 128=해수면)
  - 두 PNG는 `tools/build_world_data.py`가 생성. 크기가 같아야 함.
- **출력**: `Assets/Game/Map/WorldMapData.asset` (없으면 생성, 있으면 갱신)
- **동작**: 픽셀 1개 = 격자 셀 1개. 지형 PNG는 팔레트 최근접 분류로 `terrain[]`, 고도 PNG는 밝기값으로 `heights[]`를 채운다. PNG 좌하단 = 격자 (0,0) = 남서쪽이라 행 뒤집기는 없다.
- **사용 시점**: 최초 1회, 그리고 PNG를 수정(재변환)할 때마다 다시 실행.
- **확인**: Console에 결과 로그가 출력되고 생성 에셋이 선택·핑된다.
  예) `격자 1024x512 (524288셀) | 심해 …, 얕은바다 …, 육지 …, 산악 …`
- **코드**: `Assets/Scripts/Map/Editor/WorldMapImporter.cs`

### Build Map In Scene

현재 씬에 맵 평면과 탑다운 직교 카메라를 자동 배치해, 에디터/플레이 모두에서 세계지도를 화면에 띄운다.

- **입력**: `Assets/Game/Map/WorldMapData.asset` (없으면 먼저 Import 실행하라는 안내)
- **출력(씬 변경)**:
  - `WorldMap` 게임오브젝트 (`WorldMapRenderer` + 고도 반영 3D 지형 메시/머티리얼)
  - `Main Camera`를 직교·탑다운으로 설정(맵 위에서 똑바로 내려다봄, 북쪽=화면 위). 없으면 생성.
  - `Directional Light`를 비스듬히 배치(산맥 음영용). 그림자는 모바일 위해 off.
- **동작**: `WorldMapRenderer`가 `WorldMapData`를 읽어 ① 지형 분류 색 텍스처를 만들고 ② 고도(height)로 정점을 솟구쳐 3D 기복 메시를 생성한다. 비스듬한 광원이 산맥에 음영을 만들어 탑다운에서도 입체적으로 보인다. 비주얼은 항상 데이터에서 재생성된다.
- **성능 옵션**(`WorldMapRenderer` 인스펙터): `Height Scale`(기복 과장), `Mesh Downsample`(메시 밀도, 모바일 2~4), `Play Target Fps`(기본 30).
- **사용 시점**: 맵을 눈으로 확인할 때. 데이터 갱신(재Import) 후 다시 실행하거나, 컴포넌트의 우클릭 ▸ Rebuild로 갱신.
- **확인**: Scene/Game 뷰에 세계지도가 보이고 Console에 배치 로그가 출력된다.
- **코드**: `Assets/Scripts/Map/Editor/WorldMapSceneBuilder.cs`, `Assets/Scripts/Map/Runtime/WorldMapRenderer.cs`
