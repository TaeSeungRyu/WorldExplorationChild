## Tools ▸ World Exploration

### Import World Map (PNG → WorldMapData)

색상 코딩 PNG를 읽어 게임이 사용하는 맵 데이터 에셋(`WorldMapData`)으로 변환(굽기)한다.

- **입력**: `Assets/Game/Map/Authoring/world_terrain_512x256.png`
  (바다=파랑 `(40,90,160)`, 육지=초록 `(95,150,70)`. 변환 스크립트 `tools/geojson_to_terrain.py`가 생성)
- **출력**: `Assets/Game/Map/WorldMapData.asset`
  (없으면 생성, 있으면 갱신)
- **동작**: 픽셀 1개 = 격자 셀 1개. 각 픽셀 색을 바다/육지 팔레트 중 가까운 쪽으로 분류해 `terrain` 배열을 채운다. PNG 좌하단 = 격자 (0,0) = 남서쪽이라 행 뒤집기는 없다.
- **사용 시점**: 최초 1회, 그리고 PNG(지형)를 수정할 때마다 다시 실행해 데이터 갱신.
- **확인**: 실행 후 Console에 결과 로그가 출력되고, 생성된 에셋이 선택·핑된다.
  예) `격자 512x256 (131072셀), 육지 46224 / 바다 84848`
- **코드**: `Assets/Scripts/Map/Editor/WorldMapImporter.cs`
