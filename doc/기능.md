# 에디터 기능 (상단 메뉴)

Unity 에디터 상단 메뉴에서 실행하는 커스텀 기능 목록.
새 메뉴 기능을 추가할 때마다 이 문서에 항목을 추가한다.

---

## Tools ▸ World Exploration

### 1. Import World Map (PNG → WorldMapData)

지형 분류 PNG + 고도 PNG를 읽어 **로직용** 맵 데이터(`WorldMapData`)로 굽는다. (통행 판정·범위)

- **입력**: `Assets/Game/Map/Authoring/world_terrain_4096x2048.png`, `world_height_4096x2048.png` (`tools/build_world_data.py` 생성)
- **출력**: `Assets/Game/Map/WorldMapData.asset` (terrain[]+heights[], 4096×2048)
- **용도**: 화면 표시가 아니라 **로직**(배는 바다만/도보는 육지만 등)과 카메라 범위·위경도 변환의 기준.
- **코드**: `Assets/Scripts/Map/Editor/WorldMapImporter.cs`

### 2. Import Land Mesh (bytes → Mesh)

`tools/build_land_mesh.py`가 만든 **평면 대륙 메시**(고도 없음) 바이너리를 Unity Mesh 에셋으로 굽는다.

- **입력**: `Assets/Game/Map/land_mesh.bytes` (해안선 삼각분할 윗면만, 완전 평면 시트)
- **출력**: `Assets/Game/Map/LandMesh.asset` (~5.6만 삼각형, ~3MB, 모든 법선 +Y → 균일 음영)
- **참고**: 고도(산)는 메시에 넣지 않음 — 줄무늬·메모리 문제로 제거. 입체 지형지물은 나중에 별도 프리팹/에셋을 맵에 배치.
- **코드**: `Assets/Scripts/Map/Editor/LandMeshImporter.cs`

### 3. Build Map In Scene

현재 씬에 **단색 바다 평면 + 단색(녹색) 입체 대륙 메시 + 팬·줌 카메라 + 광원**을 배치한다.

- **입력**: `WorldMapData.asset`(범위) + `LandMesh.asset`(있으면 입체 대륙).
- **출력(씬)**:
  - `WorldMap ▸ Sea`(단색 파랑 평면) + `Land`(돌출 대륙 메시, 단색 녹색·양면). 머티리얼은 `Materials/Sea.mat`·`Land.mat`. 옛 WorldMap은 삭제 후 재생성.
  - `Main Camera`에 `MapCameraRig` — 대한민국 중심 시작, **드래그 팬 + 휠/핀치 줌**(보간).
  - `Directional Light`(그림자 off) — 돌출 대륙 옆면 음영으로 입체감.
- **색**: 단색이라 어느 줌에서도 안 깨짐. (색 변경은 `WorldMapSceneBuilder`의 `LandColor`/`SeaColor` 또는 Materials의 .mat에서)
- **비주얼 vs 로직**: 화면은 단색 메시/평면, 통행/범위는 `WorldMapData`(격자). 분리 유지.
- **조작(플레이)**: 좌클릭/한 손가락 드래그=팬, 휠/핀치=줌.
- **코드**: `WorldMapSceneBuilder.cs`, `MapCameraRig.cs`, `tools/build_land_mesh.py`
