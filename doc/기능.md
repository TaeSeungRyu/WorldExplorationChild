# 에디터 기능 (상단 메뉴)

Unity 에디터 상단 메뉴에서 실행하는 커스텀 기능 목록.
새 메뉴 기능을 추가할 때마다 이 문서에 항목을 추가한다.

## Tools ▸ World Exploration

### 1. Build Map In Scene

저폴리 월드맵 FBX를 씬에 배치하고 팬·줌 카메라 + 광원을 설정한다. **(현재 맵 비주얼의 메인 기능)**

- **입력**: `Assets/Game/Art/Models/WorldMap_LowPoly.fbx` (블렌더에서 만든 저폴리 월드맵. FBX 본래 머티리얼 사용)
- **출력(씬)**:
  - `WorldMap` — FBX 인스턴스. 임포트 시 수직이라 **`Euler(270,180,0)`로 눕힘**, 머티리얼은 **양면(Cull Off)** 처리(눕혔을 때 백페이스로 어두워짐 방지). 색은 FBX 그대로.
  - `Main Camera`에 `MapCameraRig` — 바운드 자동 계산해 전체를 비스듬히 내려다봄. **드래그 팬 + 휠/핀치 줌**. tilt/yaw/fov/줌 리셋.
  - `Directional Light`(없을 때만 추가). 환경광은 씬 기본(Skybox)으로.
- **조작(플레이)**: 좌클릭/한 손가락 드래그=팬, 휠/핀치=줌. 줌·각도는 `MapCameraRig`의 `View Height`/`Tilt Angle`로 조절.
- **주의**: FBX는 임포트 시 수직(위도=Y)이라 회전이 필요. 카메라 자동제어는 **플레이에서만**(에디트 모드에선 카메라 수동 조정 가능).
- **코드**: `Assets/Scripts/Map/Editor/WorldMapSceneBuilder.cs`, `Assets/Scripts/Map/Runtime/MapCameraRig.cs`

### 2. Import World Map (PNG → WorldMapData)  *(로직용 — 아직 미사용)*

지형 분류 PNG + 고도 PNG를 읽어 **로직용** 격자 데이터(`WorldMapData`)로 굽는다. 화면 표시가 아니라 통행 판정용.

- **입력**: `Assets/Game/Map/Authoring/world_terrain_4096x2048.png`, `world_height_4096x2048.png` (`tools/build_world_data.py` 생성)
- **출력**: `Assets/Game/Map/WorldMapData.asset` (terrain[]+heights[], 4096×2048)
- **용도(예정)**: 배는 바다만/도보는 육지만 같은 **이동 판정**(IsNavigable). 위경도→월드 = X=경도×22, Z=위도×22 (FBX와 동일 스케일).
- **코드**: `Assets/Scripts/Map/Editor/WorldMapImporter.cs`
- **현재 활성 맵(FBX)과는 분리** — 배·이동 기능을 붙일 때 사용 예정.
